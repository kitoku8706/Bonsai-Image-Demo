# Bonsai Image Demo 사용 설명서

이 문서는 Bonsai Image Demo를 처음 사용하시는 분들을 위해 작성된 쉽고 상세한 안내서입니다. 코딩이나 터미널 환경이 익숙하지 않더라도 순서대로 따라 하시면 쉽게 이미지를 생성하실 수 있습니다.

---

## 1. 시작하기 전에 (설치 및 기본 설정)

Bonsai Image Demo를 사용하려면 먼저 사용 중인 운영체제에 맞는 설정 스크립트를 한 번 실행해야 합니다.

### macOS 또는 Linux 사용자:
터미널 프로그램(Terminal)을 열고, 다운로드 받은 폴더로 이동한 후 다음 명령어를 입력하세요.
```bash
./setup.sh
```
이 명령어는 프로그램 실행에 필요한 파일과 패키지들을 자동으로 설치해 줍니다.

### Windows 사용자:
Windows에서는 **PowerShell**을 사용해야 합니다. 시작 메뉴에서 PowerShell을 검색하여 실행한 후, 아래 명령어들을 순서대로 입력하세요.
```powershell
# 1. 스크립트 실행 권한을 허용해주는 1회성 설정입니다. (PowerShell은 기본적으로 스크립트 실행을 막아둡니다)
Set-ExecutionPolicy -Scope CurrentUser RemoteSigned   

# 2. 설치 스크립트 실행
.\setup.ps1
```
*(주의: Windows에서 설치 중 오류가 발생한다면, Git이 설치되어 있는지, NVIDIA 그래픽 드라이버가 최신 버전인지 확인해 보세요.)*

---

## 2. 인공지능 모델 다운로드하기

그림을 그려줄 인공지능 '모델' 파일을 내 컴퓨터로 다운로드해야 합니다. 
모델에는 두 가지 종류가 있습니다:
- **Ternary (기본값, 추천):** 고품질의 이미지를 생성합니다. 용량이 조금 더 크지만 결과물이 좋기 때문에 가장 권장됩니다.
- **Binary:** 크기가 더 작고 가벼운 모델입니다. 컴퓨터 사양이 조금 부족할 때 시도해 볼 수 있습니다.

터미널이나 PowerShell 창에 아래 명령어를 입력하여 다운로드합니다:

```bash
# 권장 모델 (Ternary) 다운로드 - 기본적으로 이 명령어를 사용하시면 됩니다.
./scripts/download_model.sh

# (선택 사항) 더 가벼운 모델 (Binary) 다운로드
./scripts/download_model.sh binary
```

---

## 3. 웹 브라우저에서 스튜디오 사용하기 (가장 쉬운 방법 ⭐)

까만 화면에 명령어를 입력하는 것이 어색하다면, 인터넷 브라우저 창에서 마우스 클릭으로 편하게 이미지를 만들 수 있습니다.

1. 터미널(또는 PowerShell)에서 다음 명령어를 실행하여 스튜디오 서버를 켭니다.
   ```bash
   # Windows의 경우 .\scripts\serve.ps1 을 입력하세요.
   ./scripts/serve.sh
   ```
2. 서버가 켜지면 웹 브라우저(크롬, 엣지 등)를 열고 주소창에 `http://localhost:3000` 을 입력하여 접속합니다.
3. 나타나는 화면의 입력창에 만들고 싶은 이미지의 설명을 영어(또는 지원 언어)로 입력하고 버튼을 누르시면 됩니다!
4. 사용을 모두 마치고 스튜디오를 종료하고 싶을 때는, 서버를 켜두었던 터미널 창을 클릭하고 키보드에서 `Ctrl + C`를 누르시면 됩니다.

---

## 4. 터미널에서 명령어로 바로 이미지 생성하기

웹 브라우저를 띄우지 않고 터미널에서 짧은 명령어 한 줄로 바로 이미지를 만들 수도 있습니다.

### 방법 1. 웹 스튜디오가 켜져 있을 때 빠르게 생성 (추천)
위의 3번 과정을 통해 스튜디오 서버(`serve.sh`)가 실행 중이라면, **새로운 터미널 창을 하나 더 열어서** 아래 명령어로 빠르게 이미지를 만들 수 있습니다. 이 방법은 준비 시간이 짧아서 속도가 빠릅니다.

```bash
./scripts/send_request.sh -p "눈 덮인 산을 배경으로 비 내리는 숲속에 있는 얼음 분재 나무, 극사실주의 사진."
```

### 방법 2. 서버 실행 없이 1회성으로 바로 생성하기
서버를 켜두지 않고, 단 한 번만 이미지를 만들고 프로그램을 종료하고 싶을 때 사용합니다. (대신, 실행할 때마다 모델을 불러오는 준비 시간이 조금 더 걸립니다.)

**macOS / Linux:**
```bash
./scripts/generate.sh -p "원하는 이미지 설명"
```

**Windows:**
```powershell
.\scripts\generate.ps1 -p "원하는 이미지 설명"
```

## 💡 유용한 팁: 이미지 크기 조절하기

명령어로 이미지를 생성할 때 `--size` 옵션을 추가하면 이미지의 크기나 비율을 변경할 수 있습니다. 아무것도 적지 않으면 기본값인 가로세로 512x512 크기로 빠르게 생성됩니다.

* **고품질 정사각형:** `--size 1024x1024`
* **가로로 긴 사진:** `--size 1248x832`
* **세로로 긴 사진:** `--size 832x1248`

**사용 예시:**
```bash
./scripts/generate.sh -p "우주복을 입은 귀여운 고양이" --size 1024x1024
```
